What got unified

Each app contributed something. Andy-UI keeps the best canonical form of each and drops the duplicates — one button system, one table, one toast, one status language.

DevPilotcore primitives

Buttons, cards, inputs, badges, the sidebar / header / breadcrumb shell, data grid, and modal footer language — the backbone everything else inherits.

Agenticconversational

Chat bubbles, thinking accordion, streaming indicator, session list, tool-execution summary with usage meters, and tool chips.

AndySkillsregistry

Skill cards, sortable data tables with row actions, the filter toolbar, segmented grid/list toggle, namespace group headers, and empty states.

McpGatewayinfrastructure

Status pills, callouts, the page-heading + icon-chip pattern, secret/key reveal & copy boxes, and key-management tables.
